📊 Connect Google Sheets

All your leads will be stored and tracked in a Google Sheet automatically.
One-time 5-minute setup required.

1
Go to console.cloud.google.com → Create project
2
Enable Google Sheets API + Google Drive API
3
Go to IAM & Admin → Service Accounts → Create → Download JSON key
4
Rename the file to service_account.json and place it in your lead_automation folder
5
Restart python server.py — sheet is created automatically

Full guide: see GSHEET_SETUP.md in your folder
